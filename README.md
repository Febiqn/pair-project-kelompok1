# pair-project-kelompok1

### User Feature =
1.Register Membership

2.User Can Delete Membership

3.Renting PS System

4.Check Time

5.Billing System

### Admin feature =
1.View Revenue

2.Report Broken PS

3.View PS Condition

### Command you need to run

to start MySQL

`bash
sudo service mysql start
`
`bash
sudo service mysql status
`

to access MySQL

`bash
sudo mysql -u root -p
`

insert the playstations

`bash
INSERT INTO playstations (ps_name) VALUES
('PS-A'), 
('PS-B'), 
('PS-C'), 
('PS-D'), 
('PS-E'); 
`
